# CodeingAgent

## Project Overview

`codeingagent` is a Python project that implements an autonomous agent workflow using **LangChain**, **LangGraph**, and **Groq**. The system consists of a planner, architect, and coder agents that collaboratively generate and execute code based on user prompts.

## Architecture Overview

- **graph.py** – Defines the LangGraph state machine, nodes (`plannerAgent`, `architectAgent`, `coderAgent`), and the execution flow.
- **tools.py** – Provides utility tools for file operations and environment interaction used by the coder agent.
- **prompts.py** – Contains the system prompts for each agent.
- **.env** – Stores configuration such as API keys.
- **README.md** – This documentation file.

## Setup Instructions

1. **Clone the repository**
   ```bash
   git clone <repository-url>
   cd codeingagent
   ```
2. **Create a virtual environment** (optional but recommended)
   ```bash
   python -m venv .venv
   source .venv/bin/activate   # macOS/Linux
   .venv\Scripts\activate    # Windows
   ```
3. **Install dependencies**
   ```bash
   pip install -r requirements.txt
   # or, if using Poetry
   poetry install
   ```
4. **Configure environment variables**
   - Copy `.env.example` to `.env` (if provided) or create a `.env` file.
   - Add your Groq API key:
     ```
     GROQ_API_KEY=your_api_key_here
     ```

## Running the Application

The primary entry point for the agent workflow is **`graph.py`**. To execute the example prompt defined in the file:

```bash
python -m agent.graph
```

If you prefer to use the minimal `app.py` wrapper, you can run:

```bash
python app.py
```

Both commands will start the LangGraph agent and print the result of the sample user prompt.

## Example Usage

The repository includes a sample prompt in `graph.py`:
```python
user_prompt = "Create a simple calculator web app"
result = agent.invoke({"user_prompt": user_prompt})
print(result)
```
Running the script will trigger the planner, architect, and coder agents to generate and apply code steps.
